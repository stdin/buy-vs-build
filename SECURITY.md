# Security Policy

## Supported Versions

The latest released version receives fixes. buy-vs-build ships **no runtime
dependencies** and runs entirely on the Node standard library, so its surface
area is limited to the instruction files, lifecycle hooks, and benchmark scripts
in this repository.

| Version | Supported |
| ------- | --------- |
| 0.1.x   | ✅        |

## Reporting a Vulnerability

Please **do not** open a public issue for security problems.

Email **slava@mybrand.ai** with:

- A description of the issue and its impact.
- Steps to reproduce or a proof of concept.
- The affected file(s), version, or host integration.

You can expect an acknowledgement within a few business days. Once a fix is
ready, we will coordinate a disclosure timeline and credit you in the release
notes unless you prefer to remain anonymous.
